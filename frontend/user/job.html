Senior Laravel Developer

Actively Hiring
Google Inc. San Francisco, CA (Remote) Posted 2 days ago

Job Summary

We are looking for an experienced Senior Laravel Developer to join our core engineering team. You will be responsible for architecting, building, and maintaining scalable RESTful APIs and complex web applications. If you are passionate about writing clean, testable PHP code and solving complex architectural problems in a fast-paced environment, this is the perfect role for you.

Key Responsibilities

  • Design, develop, and maintain robust and scalable backend services using Laravel and PHP 8+.
  • Integrate third-party APIs and payment gateways (Stripe, PayPal, etc.).
  • Optimize database queries and overall application performance for high-traffic environments.
  • Collaborate with frontend engineers to integrate Vue.js/React UI components with APIs.
  • Mentor junior developers and conduct thorough code reviews to ensure coding standards.

Requirements & Qualifications

Required Skills:

  • Minimum 4+ years of professional experience in PHP and Laravel framework.
  • Deep understanding of RESTful API design and object-oriented programming (OOP).
  • Strong experience with MySQL/PostgreSQL and database optimization.
  • Proficiency in using Git, Docker, and CI/CD pipelines.

Preferred Skills:

  • Experience with Tailwind CSS and frontend frameworks like Vue.js or Alpine.js.
  • Understanding of AWS or DigitalOcean cloud infrastructure.

Education:

Bachelor's Degree in Computer Science, IT, or related field (or equivalent practical experience).
